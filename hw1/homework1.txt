1. What is HTML? What does it do?
HTML is the standard language for creating webpages. It provides the structure for web content by defining elements such as headings, paragraphs, links, and images.

2. What is CSS? What does it do?
CSS is used to style and format the layout of a webpage. It controls the visual presentation of HTML elements, including colors, fonts, and spacing.

4. What is an HTML element? Tag? Attribute?
An HTML element is a piece of content in an HTML document that consists of a start tag, content, and an end tag. 
A tag is used to define an element, and an attribute provides additional information about an element.

5. What are the parts of a CSS rule?
A CSS rule consists of two main parts: selector and declaration.

6. Explain how CSS rules cascade.
CSS rules cascade by following a hierarchy of specificity and order. If multiple rules apply to the same element, the one with higher specificity or that appears later in the code will override the others.
